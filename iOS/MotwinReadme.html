Readme

1. Description

This folder is exported from the Motwin IDE HTML5. It contains an XCode Project.

2. Usage

2.1. Open the project using XCode

Mac OS X is required for the following steps.

Ensure that XCode > 5.0 is properly installed.

Open the exported folder clicking on '<project name>.xcodeproj'.

2.2. Change the HTML5 code source

Some changes are required to work with PhoneGap/Cordova:

Edit index.html using a text editor.

Import cordova.js adding a script tag to the into the head tag:

<script type="text/javascript" charset="utf-8" src="cordova.js"></script>

Import cmotwin-cordova-support.js adding a script tag to the into the head tag:

<script type="text/javascript" charset="utf-8" src="path/to/motwin-cordova-support.js"></script>

Edit application.js using a text editor.

Replace the connection method from:

channel.connect();

to:

channel.useCordovaLifeCycle();
